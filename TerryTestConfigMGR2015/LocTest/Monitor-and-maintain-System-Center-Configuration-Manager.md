# 监视和维护 System Center Configuration Manager
在启动并运行 [!INCLUDE[cm6long](../LocTest/includes/cm6long_md.md)] 基础结构后，你将想要监视和维护此基础结构以使其顺畅运行。 这可以包括：  
  
-   收集清单并运行报告  
  
-   监视站点和客户端操作  
  
-   修改站点复制数据的方式和时间  
  
-   保持客户端和基础结构的修补程序和可用功能为最新  
  
-   创建备份，准备好灾难恢复方案
